# 従業員シートの列構成

## 📋 変更後の列構成（2025年10月更新）

| 列 | 列名 | データ型 | 説明 | 例 |
|---|---|---|---|---|
| A | employee_id | テキスト | 従業員ID（固定値） | EID-001 |
| B | employee_name | テキスト | 従業員名（固定値） | 山田太郎 |
| C | store | プルダウン | 店舗 | 東京, 大阪, 在宅 |
| D | role | プルダウン | 役職 | 販売, 受付, 事務 |
| E | date | 日付 | 希望日 | 2025-11-01 |
| F | start_time | 時刻 | 開始時間 | 10:00 |
| G | end_time | 時刻 | 終了時間 | 19:00 |
| H | break_hour | 数値 | 休憩時間（時間） | 1.0 |
| I | total_hour | 数値 | 総労働時間（時間）※自動計算 | 8.0 |
| J | shift_type | プルダウン | シフトタイプ | 通常, 早番, 遅番, 休み |
| K | work_content | テキスト | 業務内容 | レジ業務、商品陳列 |
| L | notes | テキスト | 備考・希望 | 学校行事の都合で... |
| M | status | プルダウン | 承認ステータス | 承認, 確認中, 非承認 |
| N | manager | テキスト | 承認者名 | 佐藤 |
| O | approved_at | 日時 | 承認日時 | 2025-10-25 14:05 |

## 🔄 変更内容

### 変更点
1. **H列**: `break_min`（分数）→ `break_hour`（時間）に変更
2. **I列**: 新規追加 `total_hour`（総労働時間・自動計算）
3. **以降の列**: 1列ずつ後ろにシフト

### 総労働時間の計算式
```
総労働時間 = (終了時間 - 開始時間) - 休憩時間（時間）
```

例：
- 開始時間: 10:00
- 終了時間: 19:00
- 休憩時間: 1.0時間
- 総労働時間 = (19:00 - 10:00) - 1.0 = 8.0時間

## 📝 データ入力例

```
EID-001	山田太郎	東京	販売	2025-11-01	10:00	19:00	1.0	8.0	通常	レジ業務、商品陳列	学校行事の都合で...	承認	佐藤	2025-10-25 14:05
```

## ⚙️ GASスクリプトの対応

GASスクリプトは新しい列構成に対応済みです：
- `break_hour`列を読み取り
- `total_hour`を自動計算
- CSVファイルに正しく出力
- マスター集約シートに正しく保存

## 🎯 運用上の注意点

1. **休憩時間の入力**: 時間単位で入力（例: 1.0 = 1時間、0.5 = 30分）
2. **総労働時間**: 自動計算されるため、手動入力不要
3. **データ検証**: H列は数値型として設定推奨
